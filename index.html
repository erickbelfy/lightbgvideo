Light Background Video

The simple implementation for run videos on your html background

There are some steps to run the class in your application.

First, insert the javascript class in your html body:

<script type="text/javascript" src="light-bg-video.js"></script>

Now you need to declare the video tag, specifying all compatible sources for the browsers

<video id="video-instance" preload="auto" autoplay loop >
												<source type="video/ogg" src="http://video-js.zencoder.com/oceans-clip.ogv">
												<source type="video/mp4" src="http://video-js.zencoder.com/oceans-clip.mp4">
												<source type="video/mov" src="http://video-js.zencoder.com/oceans-clip.mov">
										</video>

Set the html taget

<div id="video-target">...</div>

Tip: you can customize width and height of your video target. All content inside the video target needs z-index more than 1.
